The Unordered List Tag < ul >

Elements
Info

The Unordered List tag makes a list with bullet points before each item on the list

 Attributes

The < ul > tag supports the < compact > tag, which makes the list smaller than normal. And supports < type > tag which specifies the kind of marker to use in the list. Both of these tags arent supported in HTML 5.After you use the UL tag you must use the List itme tag: < li >. UL tag also supports Global Attributes and Event Attributes.

Example

Code:


< ul >

< li > potoatoes < li >

< li > Milk < li >

< li > eggs < li >

< /ul >
